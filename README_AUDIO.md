## ⚠️ 防盗链问题

经济学人的音频文件有防盗链保护，无法通过脚本直接下载。

**解决方案：**

### 方案 A：手动下载并上传（推荐）

1. **打开原文**
   - 访问：https://www.economist.com/business/2026/03/29/the-plan-to-make-ipos-great-again

2. **下载音频**
   - 在页面上找到音频播放器
   - 右键点击"下载"或"另存为"
   - 保存为 `ipo_audio.mp3`

3. **上传到微信**
   - 登录公众号后台：https://mp.weixin.qq.com
   - 进入"素材库" → "音频"
   - 点击"新增音频"
   - 上传 `ipo_audio.mp3`
   - 填写标题："The plan to make IPOs great again - Audio narration"

4. **在文章中插入音频**
   - 进入草稿箱
   - 编辑文章
   - 在副标题后插入音频素材

---
