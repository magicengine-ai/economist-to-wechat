## 🔧 下一步配置

### 1. 安装 Python 依赖

```bash
cd C:\Users\Admin\.openclaw\workspace\economist-to-wechat
pip install -r requirements.txt
```

### 2. 配置微信公众号凭证

在 `C:\Users\Admin\.openclaw\workspace\` 目录创建 `.wechat-credentials.json`：

```bash
cd C:\Users\Admin\.openclaw\workspace
copy economist-to-wechat\.wechat-credentials.example.json .wechat-credentials.json
```

然后编辑 `.wechat-credentials.json`，填入你的公众号信息：

```json
{
  "appId": "YOUR_APP_ID",
  "appSecret": "YOUR_APP_SECRET",
  "accountName": "公众号名称"
}
```

**注意**：`.wechat-credentials.json` 已添加到 `.gitignore`，不会被提交到 Git。

### 3. 测试运行

```bash
# 获取访问令牌
cd C:\Users\Admin\.openclaw\workspace\economist-to-wechat
python scripts/get_wechat_token.py

# 下载封面图（需要先配置 IMAGE_URL）
python scripts/download_cover_direct.py

# 发布文章（需要先配置文章内容）
python scripts/economist_to_wechat.py
```

2. 将翻译内容复制到 `economist_to_wechat.py` 脚本中
3. 运行脚本发布到微信公众号

### 方案 B：自动化集成

## ⚠️ 注意事项

1. **凭证安全**：`.wechat-credentials.json` 包含敏感信息，不要分享到公开仓库
2. **图片下载**：经济学人图片有防盗链，必须使用 `download_cover_direct.py` 脚本下载
3. **标题长度**：微信公众号限制标题 64 字符，脚本会自动处理
4. **格式规范**：遵循经济学人排版规范（首字母大写、结束符号等）
